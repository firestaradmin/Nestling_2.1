# Nestling_2.1
STM32F103ZET6的最小系统板及其扩展主板

最小系统板载：**用户LED、KEY、SPI-Flash、EEPROM、Type-C USB接口等**

扩展板载：**8Mbit——SRAM、ESP8266EX-WIFI、MPU6050-陀螺仪、SSD1306-OLED屏、触摸按键KEY等**
